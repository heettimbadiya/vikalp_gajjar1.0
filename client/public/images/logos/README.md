# Logo Directory

Upload your company logos here:

- `logo.png` - Main company logo for header
- `logo-white.png` - White version for dark backgrounds
- `logo-small.png` - Favicon/small icon version
- `logo-footer.png` - Footer logo variant

Recommended formats: PNG with transparent background
Recommended sizes: 
- Main logo: 200x60px
- Small logo: 32x32px or 64x64px
